# AI 长篇小说工程模板

本仓库用于创建 AI 辅助长篇小说项目。

## 推荐工具分工

- ChatGPT：总编、策划、人物和剧情审核
- QoderWork：本地逐章创作、状态文件维护
- Codex：Git、脚本、自动检查
- WorkBuddy：资料整理、宣传与办公材料

## 初始化新小说

创建新仓库后：

1. 修改 README.md。
2. 填写 `01_小说设定/小说定位.md`。
3. 填写世界观、修炼体系和人物档案。
4. 完成全书大纲与第一卷大纲。
5. 清理示例内容。
6. 创建第一次提交。
7. 暂时不要直接批量生成正文。

## 标准章节工作流

规划 → 写作 → 审核 → 修订 → 更新动态记忆 → Git 提交
