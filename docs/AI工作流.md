# AI 工作流

## 标准单章流程

1. 更新 `03_剧情规划/当前章节卡.md`。
2. 使用 `prompts/写新章节.md` 生成草稿。
3. 将草稿保存到 `05_正文/`。
4. 使用 `prompts/审核章节.md` 生成审核报告。
5. 根据审核报告修订草稿。
6. 在 `08_修改记录/` 记录修改原因。
7. 使用 `prompts/更新动态记忆.md` 更新状态。
8. 确认无冲突后整理到 `10_发布版/`。

## 每 5 章流程

- 刷新 `03_剧情规划/未来5章细纲.md`。
- 更新 `docs/剧情节奏分析.md`。
- 检查 `04_动态记忆/伏笔索引.md`。
- 检查人物状态是否有遗漏。

## 每卷流程

- 复盘分卷目标是否完成。
- 更新下一卷大纲。
- 清理废弃伏笔。
- 确认主角成长是否有阶段变化。
