## 目录说明

## 初始化顺序

1. 填写 `01_小说设定/小说定位.md`。
2. 填写 `00_规则/总体规则.md`、`00_规则/文风规范.md`、`00_规则/禁止事项.md`。
3. 填写世界观、修炼体系、势力、地图、名词表。
4. 填写主角、反派、配角和人物关系。
5. 完成 `03_剧情规划/全书大纲.md` 和 `03_剧情规划/分卷大纲.md`。
6. 写出 `03_剧情规划/未来20章粗纲.md` 和 `03_剧情规划/未来5章细纲.md`。
7. 写第一章前，填写 `03_剧情规划/当前章节卡.md`。

## 标准章节工作流

1. 更新 `03_剧情规划/当前章节卡.md`。
2. 使用 `prompts/写新章节.md` 生成草稿，保存到 `05_正文/`。
3. 使用 `prompts/审核章节.md` 生成审核报告，保存到 `07_审核报告/`。
4. 根据审核报告修订正文，修改原因写入 `08_修改记录/`。
5. 使用 `prompts/更新动态记忆.md` 更新 `04_动态记忆/`。
6. 确认无冲突后，将最终章节整理到 `10_发布版/`。
7. 每 5 章更新一次 `docs/剧情节奏分析.md`。

## 文件命名规则

- 正文草稿：`第001章_章节标题.md`
- 章节摘要：`第001章_摘要.md`
- 审核报告：`第001章_审核报告.md`
- 修改记录：`第001章_修改记录.md`
- 发布版：`第001章_章节标题.md`
